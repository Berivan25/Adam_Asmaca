# Adam Asmaca (Python Terminal Oyunu)

Bu proje, Python ile geliştirilmiş **Türkçe Adam Asmaca** oyunudur.  
Oyun tamamen terminal üzerinde çalışır ve kullanıcı dostu çerçeveli bir görünüm sunar.

---

## Özellikler

- Türkçe kelime listesi (araba, bilgisayar, masa vb.)
- Adam asmaca ASCII çizimleri
- Büyük çerçeve ile kelime ve adam çizimi gösterimi
- Yanlış tahmin sayısı ve doğru harfler gösterilir
- Tekrar oynama menüsü (oyun bitince yeniden başlatma)
- Tamamen Türkçe

---

## Gereksinimler

- Python 3.x
- Terminal / Komut Satırı
